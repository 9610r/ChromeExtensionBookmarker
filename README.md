# ChromeExtensionBookmarker
はてなブックマークへの投稿とTwitterへのツイートを同時に行う拡張機能を追加します。
またURLコピー機能を備えています。


# DEMO

下記のリポジトリと共に使う必要があります。
[9610r/BookmarkTweetFastAPI](https://github.com/9610r/BookmarkTweetFastAPI)

Chromeの拡張機能の管理からパッケージ化されていない拡張機能の読み込みにて
認識させてください。

# Installation


```bash
git clone https://github.com/9610r/ChromeExtensionBookmarker.git
```

# Usage


ex)
左クリックでコピーを押下
`https://example.com/`

Shift + 左クリックでコピーを押下
`[Example Domain](https://example.com/)`
